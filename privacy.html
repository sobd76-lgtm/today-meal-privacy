🍽️ 오늘한끼 개인정보 처리방침

최종 수정일 : 2026년 7월 5일

오늘한끼(이하 "앱")는 사용자의 개인정보를 소중하게 생각하며 관련 법령을 준수합니다. 본 개인정보 처리방침은 앱에서 수집하는 정보와 이용 목적을 안내합니다.

1. 수집하는 정보

항목 설명
사용자 ID 앱 설치 시 생성되는 고유 ID
음식 사진 사용자가 직접 업로드한 사진
식사 기록 메뉴명, 메모, 작성일 등
SNS 로그인 정보(선택) Apple, Google 등으로 로그인한 경우 계정 식별 정보

2. 개인정보 이용 목적

  • 식사 기록 저장
  • 사용자 데이터 동기화
  • 앱 기능 제공
  • 서비스 개선

3. Firebase 이용

본 앱은 Google Firebase 서비스를 이용합니다. Firebase는 데이터 저장 및 동기화를 위해 사용됩니다.

4. SNS 로그인

SNS 로그인은 선택 사항입니다. 로그인하지 않아도 대부분의 기능을 사용할 수 있습니다. 로그인 시 여러 기기에서 동일한 데이터를 사용할 수 있습니다.

5. 사진

사용자가 선택한 음식 사진만 업로드됩니다. 사진은 앱 기능 제공 외의 목적으로 사용되지 않습니다.

6. 데이터 보관

비회원 사용자는 앱 삭제 시 기존 데이터를 복구할 수 없습니다. SNS 로그인 사용자는 로그인 후 기존 데이터를 다시 사용할 수 있습니다.

7. 데이터 삭제

사용자는 앱 삭제 또는 계정 삭제를 통해 데이터 삭제를 요청할 수 있습니다.

8. 제3자 제공

개인정보를 판매하거나 제3자에게 제공하지 않습니다. 단, Firebase 서비스 제공에 필요한 범위에서 처리될 수 있습니다.
