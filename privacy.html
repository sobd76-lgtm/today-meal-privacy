🍽️ 오늘한끼 개인정보 처리방침

최종 수정일 : 2026년 7월 5일

오늘한끼(이하 "앱")는 이용자의 개인정보를 소중하게 생각하며 관련 법령을 준수합니다. 본 개인정보 처리방침은 앱에서 수집하는 정보와 이용 목적을 안내합니다.

1. 수집하는 정보

항목 내용
사용자 ID 앱 설치 시 생성되는 익명 고유 ID
음식 사진 사용자가 직접 업로드한 음식 사진
식사 기록 메뉴명, 메모, 날짜 등
SNS 로그인 정보 (선택) Apple, Google 등 로그인 시 계정 식별 정보

2. 개인정보 이용 목적

  • 식사 기록 저장
  • 기기 간 데이터 동기화
  • 앱 기능 제공
  • 서비스 안정성 향상

3. Firebase 사용

본 앱은 Google Firebase 서비스를 사용합니다. Firebase는 데이터 저장, 사용자 인증 및 서비스 운영을 위해 사용됩니다.

4. SNS 로그인

SNS 로그인은 선택 사항입니다. 로그인하지 않아도 대부분의 기능을 사용할 수 있습니다. 로그인하면 여러 기기에서 동일한 데이터를 사용할 수 있습니다.

5. 사진 업로드

사용자가 직접 선택한 음식 사진만 업로드됩니다. 사진은 앱 기능 제공 외 다른 목적으로 사용하지 않습니다.

6. 데이터 보관 및 삭제

익명 사용자는 앱 삭제 시 기존 데이터를 복구할 수 없습니다. SNS 로그인 사용자는 로그인 후 기존 데이터를 다시 사용할 수 있습니다.

사용자는 언제든지 앱을 삭제하거나 문의를 통해 계정 및 데이터를 삭제 요청할 수 있습니다.

7. 제3자 제공

개인정보를 판매하거나 제3자에게 제공하지 않습니다. 단, Firebase 서비스 제공을 위해 필요한 범위 내에서 처리될 수 있습니다.

Privacy Policy

Last Updated: July 5, 2026

Today Meal ("the App") respects your privacy. This Privacy Policy explains what information we collect and how we use it.

Information We Collect

  • Anonymous User ID generated when the app is installed
  • Meal photos uploaded by users
  • Meal records (menu, memo, date)
  • Optional social login information (Apple, Google, etc.)

Purpose of Collection

  • Provide app functionality
  • Synchronize user data across devices
  • Maintain service quality

Firebase

The App uses Google Firebase for authentication, cloud storage and data synchronization.

Optional Social Login

Signing in is optional. Users can access most features without logging in. Signing in allows data synchronization across multiple devices.

Photos

Only photos selected by users are uploaded. Photos are used solely for providing app functionality.

Data Deletion

Anonymous users may lose their data when the app is removed. Signed-in users can request deletion of their account and data.

Third-party Services

The App uses Google Firebase. No personal information is sold to third parties.
