오늘한끼 개인정보 처리방침

최종 수정일 : 2026-07-05

1. 수집하는 정보

오늘한끼는 다음 정보를 수집할 수 있습니다.

  • 앱에서 생성한 사용자 ID
  • 사용자가 업로드한 음식 사진
  • 사용자가 입력한 음식 이름 및 기록
  • SNS 로그인(선택 시)

2. 개인정보 이용 목적

  • 식사 기록 저장
  • 기기 간 데이터 동기화
  • 서비스 제공

3. Firebase 사용

본 앱은 Google Firebase를 이용하여 데이터를 저장하고 관리합니다.

4. SNS 로그인

SNS 로그인은 선택 사항입니다.

로그인하지 않아도 앱의 대부분 기능을 사용할 수 있습니다.

5. 사진

사용자가 직접 선택한 음식 사진만 업로드됩니다.

사진은 앱 기능 제공 외 다른 목적으로 사용하지 않습니다.

6. 데이터 삭제

앱 삭제 시 익명 사용자 데이터는 복구되지 않을 수 있습니다.

로그인한 사용자는 앱 내 계정 삭제 기능 또는 문의를 통해 삭제를 요청할 수 있습니다.
